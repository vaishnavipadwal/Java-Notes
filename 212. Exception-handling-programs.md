
#  **Program 1: Handle divide by zero using try-catch-finally**

```java
public class ExceptionExample {
    public static void main(String[] args) {
        int a = 10, b = 0, result;

        try {
            result = a / b;  // This will throw ArithmeticException
            System.out.println("Result: " + result);
        } catch (ArithmeticException e) {
            System.out.println("Exception caught: Cannot divide by zero.");
        } finally {
            System.out.println("Finally block always runs.");
        }

        System.out.println("Program continues after exception handling.");
    }
}
```
### **Output**

```
Exception caught: Cannot divide by zero.
Finally block always runs.
Program continues after exception handling.
```

### **Program 4: User-defined Exception for Invalid Date and Age**

```java
// Custom exception for Invalid Date
class InvalidDateException extends Exception {
    public InvalidDateException(String message) {
        super(message);
    }
}

// Custom exception for Invalid Age
class InvalidAgeException extends Exception {
    public InvalidAgeException(String message) {
        super(message);
    }
}

// Main class
public class CustomExceptionDemo {
    public static void validateDate(int day, int month, int year) throws InvalidDateException {
        if (day <= 0 || day > 31 || month <= 0 || month > 12 || year < 1900 || year > 2100) {
            throw new InvalidDateException("Date is not valid!");
        }
    }

    public static void validateAge(int age) throws InvalidAgeException {
        if (age < 0 || age > 120) {
            throw new InvalidAgeException("Age is not valid!");
        }
    }

    public static void main(String[] args) {
        int day = 32, month = 13, year = 2025;
        int age = -5;

        try {
            validateDate(day, month, year);
            System.out.println("Date is valid.");
        } catch (InvalidDateException e) {
            System.out.println("Date Error: " + e.getMessage());
        }

        try {
            validateAge(age);
            System.out.println("Age is valid.");
        } catch (InvalidAgeException e) {
            System.out.println("Age Error: " + e.getMessage());
        }

        System.out.println("Program completed.");
    }
}
```

---

### **Output**

```
Date Error: Date is not valid!
Age Error: Age is not valid!
Program completed.
```

### **Program 6: Write an application to accept the name and age of an employee from user. If age of user is less than 20 years throw user defined exception â€œInvalid Age Exceptionâ€.**

```java
import java.util.Scanner;

// Custom exception for invalid age
class InvalidAgeException extends Exception {
    public InvalidAgeException(String message) {
        super(message);
    }
}

// Main class for Employee Age Validation
public class EmployeeAgeValidation {
    
    // Method to validate age
    public static void validateAge(int age) throws InvalidAgeException {
        if (age < 20) {
            throw new InvalidAgeException("Invalid Age: Age must be 20 or older.");
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Taking user input for name and age
        System.out.print("Enter employee name: ");
        String name = scanner.nextLine();
        
        System.out.print("Enter employee age: ");
        int age = scanner.nextInt();

        try {
            // Validate the age
            validateAge(age);
            System.out.println("Employee Name: " + name);
            System.out.println("Employee Age: " + age);
            System.out.println("Age is valid for employee registration.");
        } catch (InvalidAgeException e) {
            // Handle invalid age exception
            System.out.println("Exception: " + e.getMessage());
        }

        scanner.close();
    }
}
```

```
Enter employee age: 25
Employee Name: John
Employee Age: 25
Age is valid for employee registration.
```
